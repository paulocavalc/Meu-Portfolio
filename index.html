Developer & Designer

Front End ou Back End developer são termos para diferenciar onde uma pessoa que programa acaba se especializando. Se você está entrando no mundo do Desenvolvimento, muito provavelmente classifica todo mundo como programador e programadora. O desenvolvimento web front-end é o desenvolvimento da interface gráfica do usuário de um site, por meio do uso de HTML, CSS e JavaScript, para que os usuários possam visualizar e interagir com esse site.
